## 技术架构

### 前端技术栈
- **框架**: React 18
- **构建工具**: Vite
- **UI库**: Ant Design
- **路由**: React Router
- **状态管理**: Context API
- **主题系统**: 自定义主题上下文

### 后端技术栈
- **运行时**: Node.js
- **框架**: Express.js
- **数据库**: Supabase (PostgreSQL)
- **文件存储**: Cloudinary
- **认证**: JWT

### 部署配置
- **前端部署**: Vercel
- **后端部署**: Vercel Serverless Functions
- **CI/CD**: GitHub Actions
- **域名**: 支持自定义域名
